# Simple Text Editor

### Features

1. HTML Based UI
2. Open file
3. Text justification
    - Greedy
    - DP
4. Text alignment
    - Left
    - Right
    - Justified
5. Save as file (Download the formatted text as a file)

### Technologies used

1. Simple html UI
2. Spring boot webflux backend
3. Async communication to backend through simple js

### Algorithms
1. Text justification
    - Greedy
    - Dynamic Programming
2. Text alignment
    - Simple brute force
